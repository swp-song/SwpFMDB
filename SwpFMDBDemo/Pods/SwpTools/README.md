#### 导入：

```rub

1. 手动导入：

 	* SwpTools 文件夹 导入 项目 中

	* #import "SwpTools" 			// 常用 工具类


2. CocoaPods 导入:

	* pod search SwpTools

	* pod 'SwpTools'

	* #import <SwpTools/SwpTools.h>	// 常用 工具类
```
