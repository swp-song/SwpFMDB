# SwpCateGory


#### SwpCateGory (简介):

```
封装一个常用分类, 开发使用简单方便
```
---


#### 导入：

-------

```ruby

1. 手动导入：

 	* SwpCateGory 文件夹导入项目中

    * #import "UIColor+SwpColor.h"               //  请自行引入需要使用的类目

  2. CocoaPods 导入:

    * pod search SwpCateGory

    * pod 'SwpRequest'

    * #import <SwpCateGory/UIColor+SwpColor.h>   //  请自行引入需要使用的类目

```
