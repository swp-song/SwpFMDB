### 导入

> * 手动导入：
> 
>> * **SwpPopTools** 文件夹导入项目中。
>> * **`#import "SwpPopToolsHeader.h`**
>> 
> -------

> * CocoaPods 导入:
> 
>> * **pod search SwpPopTools**
>> * **pod 'SwpPopTools'**
>> * **`#import <SwpPopTools/SwpPopToolsHeader.h>`**
>> 
> -------
